Detailed Description

Keeps track of and controls a set of ExecutionTimers.

Use for profiling.

prints a table with durations recorded by all requested timers as well as a percentage calculated with respect to a reference_timer.

the percentage is calculated using: \( P_i = \frac{\Delta t_i \times 100}{\Delta t_{reference}} \)

Parameters
timers_namesthe names of the timers which durations are to be printed.
reference_timerthe timer which duration will be used to calculate percentages.
