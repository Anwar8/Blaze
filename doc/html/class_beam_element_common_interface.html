Detailed Description

template<typename BeamSectionClass>
class BeamElementCommonInterface< BeamSectionClass >

Common interface for all beam-column elements.

This class has implementations for parts of BeamElementBaseClass interface that are common across all beam-column elements. It is a virtual class that is to be inherited by all beam-column elements, and is not meant to be instantiated on its own. That being said, it does not contain the common variables that are used by all beam-column elements, only implementations for some functions (particularly related to mapping across local and global DOFs). All variables are in BeamElementBaseClass.

calculates the global stiffness contribution of the local element and populates global_stiffness_triplets

first, the freedoms are mapped to the right size getting the number of elements of the elem_global_stiffness After that, stiffness_map is used to map where these contributions would go in the global stiffness matrix. So, this function will populate global_stiffness_triplets with sparse matrix notation

populates stiffness_map considering active and inactive DOFs for each node of the element.

Uses the position vectors of the element to do this. Does not collect row-contributions from interface nodes i.e. those not on their parent rank.

see function calc_global_stiffness_triplets, and variables stiffness_map, and global_stiffness_triplets.
