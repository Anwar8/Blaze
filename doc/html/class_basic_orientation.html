Detailed Description

creates transform matrices to shift and rotate elements

calculates the local axis of a beam-column element, and then creates a transform matrix T that will be used to transform the stiffness matrix.

◆ calc_alpha()

void BasicOrientation::calc_alpha ( coords const & origin_x)
inline

calcualte the angle between the local and global x axes

The calculation is based on: \( A \cdot B = |A||B|\cos{\theta} \) and then: \(\theta = \arccos{\frac{A\cdot B}{|A||B|}}\) Knowing that A and B are both unit vectors the multiple of their absolute value is just 1.

Parameters
origin_xthe global coord system x-axis unit vector
