◆ free_dof()

void Node::free_dof ( int dof)

activates a DoF.

Erases DoF from inactive_dofs and inserts it into active_dofs, then caculates the number of active DoFs ndof via calc_ndof.

Attention
does not care if the DoF is already active. If so, the usage of std set would take care of not having repeats in inactive_dofs and active_dofs.
Parameters
dofdegree of freedom to activate (to free).
