◆ get_num_row_contributions()

int const Node::get_num_row_contributions ( )
inline

Get the number of rows of the stiffness matrix to which this node contributes.

The calculation assumes that the self-contributions of the node to the stiffness matrix sum into the same spot for all elements connected to the node after the first one, leaving only the stiffness components that are connected to other nodes. The reason we have 2*ndof is because each element has 2 nodes.

Warning
only works assuming each element has two nodes!
Returns
int const
