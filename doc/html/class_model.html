void Model::initialise_restraints_n_loads ( )
inline

Initialise restraints and loads for the model.

This function applies the restraints to the global mesh, reducing the active freedoms. It then initialises all the loads from the load manager. Based on the restraints, the global matrices are initialised and element-stiffness mapping is established.
