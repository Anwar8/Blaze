For the first iteration: initial mapping where none of the nodes have any displacement \(\boldsymbol{U} = \boldsymbol{0}\) , and so each element \(\boldsymbol{d} = \boldsymbol{0}\) For each iteration after that, \(\boldsymbol{U} = \boldsymbol{U} + d\boldsymbol{U}\), and so \( \boldsymbol{d}\) maps from \(\boldsymbol{U}\)
